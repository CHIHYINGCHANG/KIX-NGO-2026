🚆 交通規劃

✈ Day1 關西機場 → 京都

  • 搭乘 JR Haruka 特急，約75分鐘到達京都站
  • 建議買 ICOCA + Haruka 套票（外國人限定優惠）
  • 關西機場第一航廈 B1 有JR售票機可購買

🚅 Day3 京都 → 名古屋

  • 新幹線 Nozomi 約35分鐘，Hikari 約45分鐘
  • 單程自由席約6,680円
  • 也可用 JR Pass 或 名古屋 Flexible Pass

🚌 Day6 名古屋 → 御在所岳（電車＋巴士）

  • 近鐵名古屋→近鐵四日市（急行約30分）→湯の山温泉（普通約30分）
  • 湯の山温泉駅→三重交通バス→御在所ロープウェイ前（約10分）
  • 名鐵バスセンター→名古屋湯の山高速線（直通巴士約80分）亦可
  • 高速巴士乘客可獲纜車往返割引クーポン

🚌 Day7 名古屋 → 名花之里 → 伊勢（全程大眾交通）

  • 名花之里：名鉄バスセンター（3F）→直通バス約30分，¥870/人（去程）
  • 或：近鐵名古屋→近鐵長島（約19分）→接駁バス約10分，¥710/人
  • 燈節結束後搭近鐵長島→伊勢市（特急約1小時）
  • 近鐵「伊勢志摩ライナー」直通難波，也有停伊勢市

🚌 Day8–9 伊勢・鳥羽地區（CAN BUS）

  • 三重交通「CAN BUS」觀光周遊巴士，1日券¥1,000・2日券¥2,000
  • 涵蓋外宮・內宮・二見・鳥羽水族館全線，無限搭乘
  • 內宮→鳥羽水族館直接坐CAN BUS約40分
  • 海女小屋（相差）：鳥羽バスセンター→かもめバス国崎行，約45分，¥600

🚅 Day9 伊勢/鳥羽 → 大阪

  • 鳥羽站搭近鐵→伊勢市→近鐵「伊勢志摩ライナー」→大阪難波，約2小時
  • 特急指定席¥3,060（伊勢市→難波）
  • 難波→新大阪搭地鐵御堂筋線約20分

🚇 大阪市區交通

  • 用 ICOCA 搭乘地下鐵、JR、近鐵
  • 大阪周遊卡（1或2日）可搭乘地下鐵+部分景點免費
  • Chiikawa 相關店鋪集中在梅田、心斎橋，均步行可達

💳 推薦票券

  • ICOCA 交通卡（全程通用，¥2,000押金）
  • ICOCA + Haruka 套票（關西機場限定）
  • 三重交通 CAN BUS 2日券（伊勢・鳥羽全線）¥2,000
  • 名古屋 donichi エコきっぷ（週末地鐵一日券 ¥740）
  • 鳥羽かもめバス 1日周遊券¥1,000（海女小屋當天購買）
